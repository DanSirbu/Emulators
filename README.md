# JAC - Just Another Chip-8 emulator :tada:

![Blinky](pics/blinky.png)

A [chip-8] https://en.wikipedia.org/wiki/CHIP-8 cpu emulator.

Usage:

    $ emulator ROM_FILE

Dependencies
